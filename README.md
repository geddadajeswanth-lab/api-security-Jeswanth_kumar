# 1. Consequences of exposing API key
Exposing an API key can allow unauthorized users to access the API, leading to misuse, increased costs, and potential data breaches.
It may also result in the key being revoked.

# 2. Why not log city names
Logging city names can expose user location data, which is sensitive.
According to privacy principles like GDPR data minimization, applications should avoid storing unnecessary personal data.
